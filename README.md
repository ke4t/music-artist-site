# JAK4RTA CITY - Official Artist Site

A modern, minimalistic music artist website for JAK4RTA CITY.

## Features
- **Native Audio Player:** Custom-built interactive tracklist and global persistent player.
- **Dynamic UI:** Glitch preloader, 3D tilt effects, and smooth animations.
- **Infinite Gallery:** Expanding carousel of artistic visuals.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- FontAwesome Icons
- Google Fonts (Oswald, Space Mono, Inter)

## Deployment
Deployed via Vercel and GitHub.
